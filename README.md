# thamilselvan.github.io

A personal portfolio website for Thamilselvan, a Senior .NET Engineer specializing in enterprise solutions.

## About

This static website showcases professional experience, technical skills, and contact information. Built with modern web technologies, it features a clean, responsive design with smooth animations and a professional aesthetic.

## Features

- **Hero Section**: Introduction with professional tagline and key details
- **About Section**: Personal description with professional statistics
- **Skills Section**: Organized display of technical competencies across multiple domains
- **Experience Section**: Timeline of professional roles and achievements
- **Contact Section**: Contact information and call-to-action
- **Responsive Design**: Optimized for desktop and mobile devices
- **Smooth Animations**: CSS animations and JavaScript-powered scroll reveals

## Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Custom properties, flexbox/grid layouts, animations, and responsive design
- **JavaScript**: Intersection Observer for scroll-based animations
- **Google Fonts**: Playfair Display, DM Mono, and DM Sans for typography
- **SVG**: Noise texture overlay for visual depth

## Getting Started

### Prerequisites

A modern web browser that supports HTML5, CSS3, and ES6 JavaScript.

### Viewing Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/thamil0606/thamilselvan.github.io.git
   cd thamilselvan.github.io
   ```

2. Open `index.html` in your preferred web browser.

### Deployment

This site is configured for GitHub Pages deployment. Any changes pushed to the `main` branch will automatically update the live site at `https://thamil0606.github.io`.

## Customization

To customize this portfolio for your own use:

1. Update personal information in `index.html`
2. Modify CSS variables in the `<style>` section for color scheme changes
3. Replace content in each section (About, Skills, Experience, Contact)
4. Update contact details and links
